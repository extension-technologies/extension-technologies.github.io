Environment Setup

  • Step 1: First Download and Install Node.js.

  • Step 2: After install node.js open your CMD and install expo cli.

    Copy and Paste in Command Prompt
    expo install expo-sqlite

  • Step 3: Now , Create React Native Project in Visual Studio using expo below cammand you can create new project.

    Copy and Paste in Terminal of Visual Studio
    expo init my-new-project

    You will be asked to choose a template. Press Enter to choose Blank.

  • Step 4: Run Expo Project below command, change your current dir.

    Copy and Paste in Terminal of Visual Studio
    cd my-new-project

  • Step 5: Run your project, using expo start command

    Copy and Paste in Terminal of Visual Studio
    npm start # you can also use: expo start

  • Step 6: You should see something like with a QR code generated in the terminal Download Expo client from playStore for Android & AppStore for IOS

    Download Now. Android Expo.
    IOS Expo.

  • Step 7: Final Step- As soon as the QR code got scanned, itâ€™ll automatically load and bundle/compile your project to run it in your device.

  • Alternate way: Instead of using Step 6 & 7 You can run in Your Physical Device by Turning on Usb tethering.

  • if You Want To Run In web then

    Copy and Paste in Terminal of Visual Studio
       npm run web
