Getting Started with Bluedart

Bluedart offers SOAP and REST based API's, for SOAP based API we would require zeep for python. And REST API can work with request module. SOAP request generally works with XML data, but to conveniently use them in python the zeep module works best. And the best part zeep method take JSON data as request and internally convert it to XML.

Installing Zeep#

Bluedart offers SOAP based API, to access it in python we need Zeep module.

For personal system: $ pip install lxml zeep

For ErpNext system(run cmd inside frappe-bench directory): $ ./env/bin/pip install lxml zeep

import zeepimport json
wsdl = 'http://netconnect.bluedart.com/Ver1.9/Demo/ShippingAPI/WayBill/WayBillGeneration.svc?wsdl'client = zeep.Client(wsdl=wsdl)client.service.GenerateWayBill(request_data_json, bluedart_profile_json)

To access a list of all the service methods provided by the url enter the following:

python -mzeep https://netconnect.bluedart.com/Ver1.9/ShippingAPI/Finder/ServiceFinderQuery.svc?wsdl
