Scope:

This transaction set can be used by a transportation carrier to provide shippers, consignees, and their agents with the status of shipments in terms of dates, times, locations, route, identifying numbers, and conveyance.

The Status Tracking data will be periodically extracted from the Operations ERP and using web services, the data in a Json format will be pushed to the customer web server.

There will be three retry mechanism to resend the data in case of any exception encountered during the data transfer. The exception could be of the nature of data content, network related, Infra related etc.
