Docker manual frappe

How to install & use docker in ubuntu

  1. COMMAND TO INSTALL DOCKER

apt-get install docker.io -y

  1. COMMAND TO CHECK STATUS OF RUNNING CONTAINERS

docker ps

  1. COMMAND TO CHECK STATUS OF ALL RUNNING AND STOP CONTIANERS

docker ps -a

  1. COMMAND TO CREATE NEW CONTAINER

docker run -it -d -p xx:xx --name container-name image-name --memory="1g"

-p = this annotation use for port xx:xx - port xx = before dot represent host-port xx = after dot represent container-port container-name = name of the container image-name = name of image from you want to build container

EXAMPLES TO CREATE CONTAINERS

4.1. How to create ubuntu container

docker run -it -d --name ubuntu_container ubuntu - this will create conatiner of latest ubuntu image

docker run -it -d --name ubuntu-container ubuntu:18.04 - this will create ubuntu 18.04 container

4.2 How to create erpnext container

docker run -it -d --name erpcont deep77/erp:latest

  1. How to pull image from docker-hub

docker pull image-name

for eg - docker pull ubuntu docker pull extensiontech/frappe

  1. How to check docker images

docker images

  1. How to go inside the container

docker exec -it container-name/id /bin/bash

  1. How to start stopped container

docker start conatiner-name/id

  1. How to stop running container

docker stop container-name/id

  1. How to delete running container forcefully ( for deleting container you have to stop container first but you can delete forcefully by this coomand)

docker rm -f container-name/id

  1. How to delete stop

docker rm container-name/id

  1. How to delete image

docker rmi image-name

  1. if image is in use and you want to delete

docker rmi -f image-name
