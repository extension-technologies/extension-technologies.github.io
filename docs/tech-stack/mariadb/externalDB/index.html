externalDB

In external DB: In /etc/mysql/my.cnf add: bind-address=0.0.0.0

then granting privielages on root enter mysql GRANT ALL PRIVILEGES ON . TO 'root'@'%' IDENTIFIED BY 'frappe' WITH GRANT OPTION; FLUSH PRIVILEGES; sudo service mysql restart

make sure it's accessbile from outside or the erpserver

In erpnext: bench set-mariadb-host [REMOTE-MySQL-IP/ADDRESS]

bench new-site site1.remote --db-name _a1947c1ffeceb24a --mariadb-root-username root --mariadb-root-password admin --admin-password frappe --verbose --force --install-app erpnext
